# QuestLog


QuestLog

A Flask app that turns your todo list into quests that you complete to gain XP.
Built with Python and Flask, QuestLog gamifies your tasks so completing them feels
like leveling up in a game.

Features
- Turn todo items into quests
- Gain XP for completing quests
- Track progress in a simple UI

Tech Stack
- Python
- Flask
- HTML / CSS

Installation
1. Clone the repository
2. Create a virtual environment
3. Install dependencies with pip install -r requirements.txt
4. Run the app with flask run

Usage
Add quests, complete them, and earn XP.
